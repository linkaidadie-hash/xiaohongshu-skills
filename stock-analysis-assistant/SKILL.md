# A股分析助手

## 核心功能

1. **涨停板分析**：封单量、封单比例、涨停时间
2. **龙头股识别**：连板数量、市场情绪判断
3. **板块热点**：行业板块涨跌排行
4. **选股策略**：基于资金流、技术面、财务数据

## 数据获取

使用 mx-data skill 获取东方财富数据：
- 涨停板数据：`mx_data涨停板(limit)`
- 资金流向：`mx_data板块资金流()`
- 个股行情：`mx_data行情(股票代码)`

## 涨停板分析框架

### 核心指标
| 指标 | 说明 | 优质 |
|------|------|------|
| 封单金额 | 收盘封单量 | >5000万 |
| 封单比例 | 封单/成交额 | >30% |
| 涨停时间 | 首次封板时间 | <10:00 |
| 连板数 | 连续涨停天数 | 3板以上 |

### 风险信号
- 尾盘炸板（14:30后开板）
- 封单持续减少
- 大盘走弱时逆势封板

## 选股标准

### 激进型（高风险高收益）
- 昨日涨停，今日高开
- 板块龙头，跟随热点
- 流通市值 < 50亿

### 稳健型
- 趋势向上的涨停股
- 换手率 5-15%
- 板块有持续催化

## 风险控制

1. 单只仓位不超过20%
2. 止损线：-7%必须出
3. 盈利线：+15%可考虑减仓
4. 每天收盘前30分钟检查持仓
