# 小红书助手

## 核心功能

1. **文案生成**：标题、正文、标签
2. **封面图生成**：用 MiniMax API 生成图片
3. **话术模板**：引流、成交、私信话术

## 封面图生成

用 MiniMax image-01 模型生成小红书封面：

```bash
curl -s -X POST "https://api.minimaxi.com/v1/image_generation" \
  -H "Authorization: Bearer $MINIMAX_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "image-01",
    "prompt": "中文提示词，描述封面风格、颜色、文字内容",
    "aspect_ratio": "3:2"
  }'
```

## 文案结构

**爆款标题公式**：
- 数字型：「XX个方法/技巧/真相」
- 痛点型：「为什么你XXX却XXX」
- 结果型：「XX天，XX结果」

**正文模板**：
- 开头：引发共鸣的场景/痛点
- 中间：具体方法/步骤（分点写）
- 结尾：引导互动（评论/关注）

## 内容方向

- **副业赚钱粉**：AI工具、变现方法、避坑指南
- **宝妈粉**：育儿经验、兼职赚钱、生活记录
- **职场粉**：职场技巧、副业转型、效率工具

## 标签策略

每篇笔记加 5-10 个标签：
- 1个大流量标签（10万+）
- 3-4个中流量标签（1万+）
- 2-3个精准标签（千级）
