# 小红书内容创作助手

## 核心功能模块

### 1. 文案生成系统
**核心能力**：标题、正文、脚本、话术全覆盖

#### 爆款标题公式
| 类型 | 公式 | 示例 |
|------|------|------|
| 数字型 | 「数字+方法/技巧」 | 「5个AI工具，月入10k」 |
| 痛点型 | 「为什么你XXX却XXX」 | 「为什么你努力却还是穷」 |
| 结果型 | 「时间+结果」 | 「30天，我从0粉到10w」 |
| 反问型 | 「你知道XXX吗」 | 「你知道这个副业赚钱方式吗」 |
| 对比型 | 「XXX vs XXX」 | 「AI工具 vs 传统软件」 |
| 秘密型 | 「大多数人不知道」 | 「大多数人不知道的赚钱秘密」 |
| 极端型 | 「最XXX的」 | 「最快的变现方法」 |

#### 正文结构模板
```
【开头】（5秒内吸引）
- 戳痛点：描绘理想vs现实的落差
- 引共鸣：用数据/案例说话
- 制造悬念：「但有个办法...」

【中间】（步骤化解决）
✓ 第1步：具体做什么 + 为什么
✓ 第2步：具体操作方法 + 技巧
✓ 第3步：常见误区 + 避坑指南
✓ 第4步：效果预期 + 时间周期

【结尾】（引导互动）
- 号召行动：「点赞→收藏→评论」
- 反向激励：「你是第几个发现这个的」
- 互动问题：「你还有其他方法吗」
- 引导关注：「关注不迷路」
```

#### 文案风格指南
- **目标受众**：宝妈、职场人、学生、创业者
- **语气**：亲切、实用、有趣、不高冷
- **长度**：800-1500字（黄金比例）
- **排版**：短句为主，分点呈现，适度使用emoji
- **逻辑**：因果关系清晰，循序渐进

### 2. 智能封面图生成

#### 使用MiniMax API生成
```bash
curl -s -X POST "https://api.minimaxi.com/v1/image_generation" \
  -H "Authorization: Bearer $MINIMAX_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "image-01",
    "prompt": "【中文提示词】小红书封面，风格描述+颜色+文字+情绪",
    "aspect_ratio": "3:2"
  }'
```

#### 高效能提示词公式
```
【主体】+ 【风格】+ 【色调】+ 【情绪】+ 【文字建议】

示例：
"年轻女性使用笔记本电脑工作，现代简约风格，清新蓝绿色调，积极向上的氛围，标题文字'30天月入10k'，小红书封面"
```

#### 封面设计要点
| 维度 | 建议 |
|------|------|
| **尺寸** | 3:2（1200×800px最佳） |
| **文字** | 3-5个关键词，大号标题 |
| **颜色** | 2-3个主色调，对比度高 |
| **人物** | 真人效果>卡通，表情生动 |
| **背景** | 简洁不复杂，突出主题 |
| **排版** | 避免满版文字，留白充足 |

### 3. 账号定位与策略

#### 定位诊断框架
```
【领域选择】
 ↓
【目标用户】（精准画像）
 ↓
【内容柱】（3-5个垂直领域）
 ↓
【差异化角度】（你的独特优势）
 ↓
【变现路径】（电商/广告/课程/服务）
```

#### 内容柱设定示例
| 内容柱 | 占比 | 频率 | 示例 |
|------|------|------|------|
| 干货分享 | 40% | 2-3次/周 | 工具教程、方法论 |
| 案例拆解 | 30% | 1-2次/周 | 成功案例、失败教训 |
| 生活分享 | 20% | 1次/周 | 日常、故事、观点 |
| 互动话题 | 10% | 1次/周 | 问卷、投票、讨论 |

#### 内容策略优化
- **选题来源**：热搜×评论×同赛道热文
- **发布时机**：工作日9-11点、18-20点（数据支撑）
- **互动运营**：前30分钟是关键，主动回复热评
- **数据复盘**：周度分析浏览、互动、保存、分享率

## 常见问题

**Q: 多久能起号？**  
A: 15天冷启动期，坚持发优质内容，第3周通常能看到数据增长。

**Q: 粉丝少怎么办？**  
A: 优化标题和封面、提升互动率、定期参加话题挑战。

**Q: 怎么找选题？**  
A: 看热搜、看同赛道爆文、看评论区问题。

**Q: 什么时候变现？**  
A: 有1000粉+50赞均值后可考虑，但建议积累3000粉再变现。
