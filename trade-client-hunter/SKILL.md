# 外贸B2B获客全能助手

## 核心功能模块

### 1. 开发信撰写系统

#### 开发信模板库

##### 模板1：价值聚焦型（首次开发）
```
Subject: [Your Name] - [具体产品名] Solution for [客户行业]

Hi [Prospect Name],

I came across [Company Name] and was impressed by [具体业务/成就].

We specialize in [你的产品/服务核心价值], and have helped [数字] companies 
in [行业] achieve [具体成果，e.g., 20% cost reduction].

I think we could bring similar results to your operation, particularly with 
[客户的具体挑战].

Would you have 15 mins this Thursday or Friday for a quick conversation?

Best regards,
[Your Name]
[Job Title]
[Company]
[Phone]
[Email]
```

##### 模板2：问题导向型（冷开发）
```
Subject: Question about [客户公司名] - [具体问题]

Hi [Name],

I have a quick question about your [产品/服务] strategy...

We work with [行业] companies and frequently hear that [痛点问题]. 

I noticed [客户的具体情况], and wondered if this is something you're 
looking to address?

If it is, I've got an idea that might help. If not, no worries.

Are you the right person to chat with about this?

Thanks,
[Your Name]
```

##### 模板3：推荐介绍型（有引荐）
```
Subject: Intro: [Your Name] + [Prospect Name]

Hi [Name],

[推荐人名字] thought we should connect. He mentioned you're looking for 
[客户需求].

We've been helping [行业] companies optimize their [具体流程], saving 
them around [数字/百分比] in costs.

Happy to share how we do it if you're interested.

Best,
[Your Name]
```

##### 模板4：新品上市型
```
Subject: New [产品名] - Exclusive Early Access

Hi [Name],

We just launched [新产品], and I thought of you immediately because 
[客户的具体原因].

Early customers are seeing [结果], and we're offering [特殊条件] 
to a select few.

Interested in learning more?

[Your Name]
```

#### 开发信黄金法则
| 要素 | 要求 | 示例 |
|------|------|------|
| **主题行** | 15字以内，清晰value point | ❌ "Hi" ✓ "10% Cost Cut for [Company]" |
| **开头** | 3秒戳中痛点或价值 | ❌ "我们是一家公司" ✓ "I noticed you..." |
| **中间** | 数字说话，避免文案腔 | ❌ "我们很专业" ✓ "Helped 50+ companies" |
| **CTA** | 具体时间，不要"有空聊" | ❌ "欢迎联系我" ✓ "15 mins Thu/Fri?" |
| **签名** | 完整信息，专业邮箱 | ✓ 名字+职位+电话+邮箱 |

#### 开发信发送策略
```
【第1天】发送首封开发信
  ↓ 【3-5天】无回复，发送跟进1
  ↓ 【5-7天】无回复，发送跟进2（换角度）
  ↓ 【7-10天】无回复，改为电话或LinkedIn
  ↓ 【30天】入pool，定期触达（月1次）
```

### 2. 客户画像与分层系统

#### 目标客户分析框架
```
【公司层面】
 ├─ 公司规模（员工数、营收）
 ├─ 行业分类
 ├─ 地理位置
 └─ 采购预算

【决策层面】
 ├─ 决策者职位
 ├─ 采购流程（多方决策？）
 ├─ 采购周期
 └─ 购买频率

【需求层面】
 ├─ 核心痛点
 ├─ 产品用途
 ├─ 预算范围
 └─ 技术要求

【机会层面】
 ├─ 市场增长
 ├─ 竞争格局
 └─ 季节性
```

#### 客户画像详细模板
| 维度 | 说明 | 示例 |
|------|------|------|
| **公司规模** | 年营收/员工数 | 200万美元/50人 |
| **行业** | 所属产业 | 制造业-金属加工 |
| **地区** | 主要市场 | 德国、意大利、西班牙 |
| **年采购额** | 预期购买量 | $50-200k/年 |
| **采购频率** | 购买周期 | 季度采购/持续供应 |
| **产品用途** | 应用场景 | 工业零部件 vs 消费品 vs B2B贸易 |
| **决策人** | 采购权力 | 采购部经理 vs CEO vs 工程师 |
| **竞争对手** | 当前供应商 | A厂商、B厂商 |
| **pain points** | 当前困扰 | 交货期长、价格高、品质不稳 |
| **潜力评分** | 优先级 | 1-5星（根据采购力+需求匹配度） |

#### 客户分层模型
```
【VIP客户】（年采购额>50万美元）
 → 专属销售、高频跟进、定制方案、邀请参访

【A类客户】（年采购额10-50万）
 → 定期拜访、定制报价、关系维护

【B类客户】（年采购额1-10万）
 → 标准跟进、模板方案、邮件为主

【C类客户】（试探性询价）
 → 自动化跟进、观察转化、低成本维护
```

### 3. 多平台获客策略

#### LinkedIn获客攻略
```
【第1步】精准找人
- 搜索公司 → 筛选部门（Procurement/Sourcing）
- 使用高级搜索：Industry + Job Title + Company
- 筛选条件：活跃用户（最近3个月有更新）

【第2步】Connection策略
✓ 先看资料，了解其职位和背景
✗ 不要带InMail消息（显得很生硬）
✓ 3-5天后无回应再发消息

【第3步】消息文案
"Hi [Name], I come across your profile and noticed you're in [职位]. 
We work with similar companies in [行业]. Would love to chat and explore 
if there's an opportunity to work together. [1句简介你的价值]"

【第4步】后续维护
- Connection后3-5天再发消息
- 定期互动对方内容（评论、点赞）
- 每月1-2条有价值的内容分享
```

#### 阿里国际站RFQ获客
```
【报价黄金法则】
✓ 24小时内回复（越快越好）
✓ 清晰填写所有字段
✓ 附图片或视频（3D优先）
✓ 给出价格区间和MOQ
✓ 注明交货期和支付条款

【报价模板】
Product Specification: [详细规格]
Price: $XX - $XX per unit
MOQ: [最小起订量]
Lead Time: [交货天数] days
Packaging: [包装]
Payment Terms: TT 30%/70% or 50%/50%
Certificates: [认证信息]

【高效能话术】
"Appreciate the inquiry. To provide accurate pricing, could you clarify...
[2-3个关键问题]"
```

#### Facebook/Google广告策略

### 4. 询盘-报价-谈判-成交全流程话术

#### 询盘回复话术库
| 场景 | 话术示例 |
|------|---------|
| **首次回应** | "Thank you for the inquiry! Before I send the quotation, could you help me understand... [关键问题]" |
| **明确需求** | "Just to confirm - you need [产品] in [规格/数量/时间]? That's perfect, we can deliver." |
| **突出优势** | "We specialize in this exactly. Compared to standard suppliers, we offer [差异化优势]." |
| **加快进度** | "I'd like to send you a formal quote today. Could you confirm [核心需求] so I get it 100% right?" |

#### 报价话术
```
【报价前的铺垫】
"Before I send the quote, let me make sure I understand your needs correctly:
- Specifications? 
- Order quantity and timeline?
- Any specific certifications needed?
- Budget range?"

【报价确认】
"Here's our proposal for [产品]. This pricing is valid for [30] days.
Key highlights:
✓ [价格点]
✓ [交货期]
✓ [质量/认证]
✓ [支付条款]

Questions?"

【应对砍价】
"I appreciate your business. To improve the pricing, 
could you increase the order volume? That would help us [降成本原因]."
```

#### 谈判场景话术
| 谈判场景 | 最佳应对 |
|---------|---------|
| **价格太高** | "What specific price point did you have in mind? And if we can reach that at higher volumes, let's see the numbers." |
| **要求降价** | "Understand. Our margins are already optimized, but if you commit to [数量/周期], we can improve the rate." |
| **竞品更便宜** | "Different specs and quality often drive the price difference. Can you share their quote? Let me see if we're comparing apples to apples." |
| **要求延期付款** | "For larger orders, we can offer D/P terms. For first cooperation, we typically do TT or L/C." |
| **质量疑虑** | "We have [认证/证书], and can send samples. Also, here are 3 reference customers you can contact." |

#### 成交催化话术
| 阶段 | 话术 |
|------|------|
| **一周跟进** | "Just checking in on the proposal. Any questions I can help clarify?" |
| **两周跟进** | "Have you had a chance to review? Happy to address any concerns or negotiate terms." |
| **三周跟进** | "Just want to confirm - are we moving forward with this order? What would help finalize it?" |
| **四周跟进** | "We have stock reserved for you until [日期]. After that, availability might be tight. Ready to place the order?" |

#### 成交信号识别
```
✓ 客户开始询问细节（交货日期、包装、支付方式）
✓ 要求发正式发票或ProForma Invoice
✓ 提出担保或样品要求
✓ 询问可用库存
✓ 提到"采购部"或"财务部"

【当看到这些信号时】
→ 立即准备：正式报价单、样品、认证文件
→ 主动确认：所有技术细节
→ 敲定条款：付款条件、交货方式
```

### 5. 长期关系维护框架

#### 订单后维护
```
【订单确认后】
- 24小时内发送订单确认
- 提供Tracking number和预计交货
- 周度进度更新

【发货前】
- 质检照片/视频给客户
- 确认最终细节

【发货后】
- 发送完整物流信息
- 7天后主动跟进：收货确认
- 14天后：满意度反馈

【交付完成】
→ 发送客户反馈问卷
→ 邀请参加VIP客户计划
```

#### 复购引导
```
【第1个订单】→ 建立信任期
  ├─ 质量过硬
  ├─ 及时沟通
  └─ 专业交付

【3-6个月】→ 第二次接触
  ├─ 主动推送新产品
  ├─ 邀请参观工厂
  └─ 提供优先库存

【首次复购成功】→ VIP计划
  ├─ 专属客服
  ├─ 优先价格
  ├─ 灵活付款
  └─ 定期业务回顾
```

#### 客户retention话术
| 情景 | 话术 |
|------|------|
| **长期合作邀请** | "We've really enjoyed working with you. I'd love to set up a regular supply agreement. Would monthly/quarterly contracts work for you?" |
| **VIP计划邀请** | "You're one of our best customers. We'd like to offer you [特殊优惠/优先权]. Are you interested?" |
| **风险预警** | "I noticed your recent orders are lower. Is everything okay? Any issues we can help with?" |
| **节日问候** | "Just wanted to reach out during [节日] to thank you for the partnership. Looking forward to more collaboration." |

## 行业最佳实践

### 高效能话术总结
```
【不要说】                    【改成这样】
"我们很专业"          →      "我们为30+国家供货"
"欢迎询价"            →      "这个产品我们有库存"
"质量保证"            →      "通过ISO 9001认证"
"价格有竞争力"        →      "$XX/PC，比市场便宜20%"
"感谢您的询价"        →      "感谢您的询价，24小时内报价"
```

### 常见陷阱
- ❌ 报价前不确认规格（浪费时间）
- ❌ 忽视客户follow-up问题（显得不专业）
- ❌ 承诺做不到的交货期（破坏信任）
- ❌ 报价没有有效期（陷入长期谈判）
- ❌ 不跟进已成交客户（失去复购机会）

## 常见问题

**Q: 开发信的回复率多少才算正常？**  
A: 冷开发回复率1-3%属于正常。如果低于1%，检查：主题行、时机、目标客户精准度。

**Q: 多久应该跟进一次？**  
A: 第1次3-5天，第2次7-10天，第3次14天。之后可转入月度newsletter。

**Q: 海关数据哪个平台最准？**  
A: Trade Data Monitor和Panjiva数据较全。中国海关数据可用于反向查询进口商。

**Q: 如何提高报价转单率？**  
A: 1) 确保规格100%正确 2) 报价附送样品 3) 清晰表达优势 4) 给出有期限的报价（30天）

**Q: VIP客户如何维护？**  
A: 月度电话会议、季度访厂邀请、优先库存、灵活支付条件。
