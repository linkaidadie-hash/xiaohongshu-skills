# 外贸获客助手

## 核心功能

1. **开发信撰写**：多场景英文/小语种开发信
2. **客户画像**：目标客户特征分析
3. **话术模板**：询盘回复、跟进、报价话术
4. **平台获客**：LinkedIn、阿里国际站、独立站

## 开发信模板

### 首次开发信
```
Subject: [产品名] from [你的公司]

Hi [Name],

I noticed [怎么发现客户] and thought of you.

[一句话介绍你能帮客户解决什么问题].

We've helped companies like [同类客户] achieve [具体结果].

Would you be open to a 5-min call this week?

Best,
[你的名字]
```

### 跟进邮件
```
Subject: Re: [产品名] - Quick follow up

Hi [Name],

Just circling back on my last email.

[一个问题或新的信息].

Happy to share some relevant case studies if helpful.

Best,
[名字]
```

## 客户画像模板

| 维度 | 填写 |
|------|------|
| 公司规模 | 小/中/大 |
| 年采购额 | $ |
| 采购频率 | 月/季/年 |
| 目标市场 | 国家/地区 |
| 产品用途 | 工业/消费/批发 |
| 决策人 | 采购/老板/工程师 |

## 平台获客策略

### LinkedIn
- 找对口公司的采购/owner
- 发connection request时不要带"InMail"
- 通过后等3-5天再发消息

### 阿里国际站
- RFQ报价要快（24h内）
- 报价要包含：价格区间、MOQ、交货期
- 附上产品视频或图片

### 独立站
- Google SEO关键词
- Facebook广告定向

## 报价模板

```
Product: [产品名]
Price: $XX - $XX/PC
MOQ: XX PCs
Lead time: XX days
Payment: TT 30%/70%
Package: [包装详情]

Validity: 30 days

Thanks,
[名字]
```

## 常用话术

### 询盘开场
"Great inquiry! Before I send the quotation, could you help me with a few quick questions..."

### 讨价还价
"We appreciate your business. To give you a better price, could you increase the order quantity?"

### 催单
"Just wanted to check if you have any updates on the PO? We have availability for [日期]."
