# MiniMax Skills 合集

面向中文用户的小红书运营、股票分析、外贸获客助手。

## Skills

### xiaohongshu-assistant
小红书内容创作助手。生成文案、封面图、内容策略、引流话术。

### stock-analysis-assistant
A股短线分析助手。涨停板分析、龙头股识别、选股策略、风险预警。

### trade-client-hunter
外贸获客助手。开发信模板、客户画像、平台获客策略、报价话术。
